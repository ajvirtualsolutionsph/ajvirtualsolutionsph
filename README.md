# AJ Virtual Solutions — Portfolio Website

A professional portfolio for AJ Virtual Solutions, showcasing services in AI Automation, Bookkeeping, and Admin Assistance. Built with plain HTML, CSS, and JavaScript — no frameworks, no build step.

## Tech Stack

- HTML5 · CSS3 · Vanilla JavaScript
- Deployed via [Vercel](https://vercel.com) (GitHub auto-deploy)
- Domain via [Namecheap](https://namecheap.com)

## Run Locally

No server needed — just open the file:

```
open index.html        # macOS
start index.html       # Windows
xdg-open index.html    # Linux
```

Or drag `index.html` into any browser.

## Project Structure

```
├── index.html
├── styles.css
├── script.js
├── assets/
│   ├── images/        # Profile / hero images
│   └── projects/      # Project screenshots
└── README.md
```

## Deployment

1. Push to GitHub
2. Import repo in [Vercel](https://vercel.com/new)
3. In Namecheap DNS, point your domain to Vercel
4. Add the custom domain in Vercel project settings
